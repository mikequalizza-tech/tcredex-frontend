## Overview

tCredex takes data security seriously. Your project information, documents, and personal data are protected using industry-standard security practices.

---

## How We Protect Your Data

### Encryption

**In Transit:**
- All data transmitted between your browser and tCredex uses TLS 1.3 encryption
- This is the same level of security used by banks and financial institutions

**At Rest:**
- Documents and project data are encrypted when stored
- Encryption keys are managed securely and rotated regularly

### Authentication

**Secure Login:**
- tCredex uses Clerk for authentication, a leading identity platform
- Support for strong passwords and multi-factor authentication (MFA)
- Session management with automatic timeouts

**Access Control:**
- Role-based permissions ensure users only see what they should
- Sponsors see their own projects
- CDEs and Investors see only projects they've been matched with or invited to

---

## What Data We Collect

### Account Information

- Name and email address
- Organization details
- Role (Sponsor, CDE, Investor)

### Project Data

- Project details entered in the Intake Form
- Documents you upload
- DealCard information

### Usage Data

- How you navigate the platform (for improving user experience)
- Feature usage (anonymous and aggregated)

---

## What We Don't Do

### We Do Not:

- **Sell your data** - Your information is never sold to third parties
- **Share without permission** - Project data is only shared with parties you've matched with
- **Use documents for marketing** - Your files are yours
- **Store unnecessary data** - We only keep what's needed for the platform

---

## Document Security

### Uploads

- Documents are scanned for malware before storage
- Files are stored in secure cloud infrastructure
- Only authorized users can access your documents

### Closing Room

- Documents in the Closing Room are access-controlled
- Only parties invited to the deal can view files
- All access is logged for audit purposes
